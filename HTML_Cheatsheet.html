Html Cheatsheet

Tag Name Description
<p> Paragraph Used for large chunks of text
<h1> Header Used for headings. Decrease in size from 1-6
<strong> Bold Used to make the enclosed text bold. Can be replaced with CSS styling
<emp> Emphasis Used to make enclosed text italics
<div> Division Used as a container element
 

CSS Cheatsheet

Property Description
font-family Sets the font of the selected item
font-weight Sets how bold the text appears on a scale of 100-900
text-align Sets the alignment of the text in the container
border Creates a border around the container. Type and thickness can be changed to you needs/wants
margin Sets how much space is on the outside of the container
